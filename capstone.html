The initial state of the solution has a working code of the application written using Python and Django web framework, which also includes unit testing. In this capstone, I faced challenges to improve the web application and deploy it into a containerized environment through a continuous integration and continuous delivery (CI/CD) pipeline.

The developer makes updates to application code using an AWS supported IDE. Code changes are pushed to a Git repository, initiating a new CI/CD pipeline in AWS CodePipeline. The pipeline uses AWS CodeBuild to run unit tests and build container images, which are then deployed to Amazon EKS cluster.
