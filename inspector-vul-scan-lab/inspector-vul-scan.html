Amazon Inspector Lab

Vulnerability Scanning & Security Assessment

Utilize Amazon Inspector to scan for vulnerabilities in AWS resources, specifically Amazon EC2 instances and AWS Lambda functions. Learn how to configure Amazon Inspector, interpret vulnerability reports, and remediate security findings.

Lab Objectives

Duration: ~75 minutes

By the end of this lab, you should be able to:

  • Activate and configure Amazon Inspector
  • Analyze and interpret vulnerability findings
  • Remediate the vulnerabilities found by Amazon Inspector

AWS Services Used

Amazon Inspector

A vulnerability management service that continuously scans your AWS workloads for software vulnerabilities and unintended network exposure. Amazon Inspector automatically discovers and scans running Amazon EC2 instances, container images in Amazon ECR, and AWS Lambda functions.

AWS Lambda

A compute service that runs your code in response to events. It automatically manages compute resources, enabling you to build applications that respond quickly to new information with automatic scaling and resource management.

Amazon EC2

Provides on-demand, scalable computing capacity in the AWS Cloud. Using Amazon EC2 reduces hardware costs so you can develop and deploy applications faster with configurable security, networking, and storage options.

AWS Systems Manager

Gives you visibility and control of your AWS infrastructure. It provides a unified interface to view operational data from multiple AWS services and helps manage your infrastructure securely at scale.

Lab Scenario

The developers at AnyCompany Startup are in the initial phases of building an application primarily using AWS Lambda. Throughout the development process, they need an automated security tool that not only scans for vulnerable software packages, but also scans within the code itself. You decide to utilize Amazon Inspector to fill this need.


Amazon Inspector meets the requirements of being able to scan AWS Lambda functions by quickly responding to new deployments. It also automatically scans additional resources such as EC2 instances within AnyCompany's AWS account.
