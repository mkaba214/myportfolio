Sample Lambda Function

import json import boto3 from datetime import datetime def lambda_handler(event, context): try: # Process incoming event body = json.loads(event['body']) if 'body' in event else event # Business logic here result = { 'timestamp': datetime.utcnow().isoformat(), 'processed': True, 'data': body } return { 'statusCode': 200, 'headers': {'Content-Type': 'application/json'}, 'body': json.dumps(result) } except Exception as e: return { 'statusCode': 500, 'body': json.dumps({'error': str(e)}) }

Architecture Highlights

  • Serverless: No server management required
  • Scalable: Automatic scaling based on demand
  • Cost-effective: Pay only for execution time
  • Event-driven: Responds to various AWS service triggers
